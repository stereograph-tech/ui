Listview (single-line)

Introduction

The list view (part of the .standardGrid class) will be contained in StandardGrid.css and is still in progress. There are some core concepts to understand, however, when implementing a listview.

Example



    <section role="main" class="row">
        
        <div class="col">
        
            <!-- in a list-view only page, you simply need a 
                single div tag with a ".col" class applied.
                You include your listview inside of this column as
                usual. Meaning, you can define your listview
                in a separate file, and dynamically include it. -->
                
            <div class="listview">
                <header class="row height3">
                    <div class="ratio2 pad"><h6>COL HEADER</h6></div>
                    <div class="ratio pad"><h6>COL HEADER</h6></div>
                    <div class="ratio pad"><h6>COL HEADER</h6></div>
                    <div class="ratio pad"><h6>COL HEADER</h6></div>
                    <div class="ratio2 pad"><h6>COL HEADER</h6></div>
                </header>
                <div class="row height4">
                    <div class="ratio2"><h4>contents</h4></div>
                    <div class="ratio pad"><h4>contents</h4></div>
                    <div class="ratio pad"><h4>contents</h4></div>
                    <div class="ratio pad"><h4>contents</h4></div>
                    <div class="ratio2 pad"><h4>contents </h4></div>
                </div>
            </div>
            
        </div>
        
    </section>

Core Concepts

The .ratio{N} class

.ratio{N} classes determine the horizontal stretch factor of each column in a listview. A .ratio3 class will stretch a column to 30% of the width of the listview, .ratio5 will stretch to 50% of the width of the listview, and so on. .ratio by itself (with no number), will cause the column to stretch to exactly 1 fraction of the available width. Mathematically, if 5 columns are in a listview, each with a .ratio class, then each column will calculate to be 20% wide. Since there is usually a mixture of different ratios within a listview, the fixed-size ratios are calculated first, subtracted from the 100%, and the remainder width is then divided evenly by any columns with the basic ratio class applied.

Defining a listview

You only need to define a single header row and 1 single content row. A data-binding loop will take care of inserting the remaining rows at runtime.
