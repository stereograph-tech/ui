Panoramic with Flex

Introduction

A panoramic view is one that is unbroken and extensive, as of a landscape. In the newest apps, this is a common layout, which involves positioning page elements in multiple columns, with horizontal scrolling to handle overflow.

How to use:

The important detail is to use minWidth{N} classes on each column. Otherwise, the columns will not be able to extend past the width of the viewport (they will simply flex to fit within it). A min width ensures the columns will be at least at a width that would enable the panorama to work.

It's still really easy to implement: 


    <section role='main' class='panoramic'>
	
        <!-- min width of 10GU -->
        <div class='col minWidth10'>Column 1</div>
		
        ... columns with minWidth{N} as desired...

        <!-- min width of 10GU -->
        <div class='col minWidth20'>Column N</div>
		
    </section>
