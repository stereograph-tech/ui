The mighty EM

Introduction

An EM, unlike px or % is a sizeless unit, meaning that it by itself does not have a set value. Instead, it inherits its value from a font-size. That is because the "EM" literally refers to the width of the capital letter "M" in the current font (roughly speaking). The EM originated in the old days of wood-block typesetting.

How It Works

As you increase or decrease the font-size, the value of the EM increases or decreases as well. This means that you can control the entire grid size of everything inside of the <body> by changing 1 simple value.

Where It's Applied

Layout.css is where we define our base font-size (and grid unit size). It looks something like this:


    body {
        font-size: 10px; 
    }

The examples on the right show the effect on the entire layout by changing the body font-size.
