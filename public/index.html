Blueprint

Introduction

Blueprint is an intuitive, flexible tool to align all of the contents of a layout to an underlying grid. It is similar to other well-known CSS Frameworks, such as 960 Grid and Bootstrap in that it provides a library of CSS classes to help create an adaptive and responsive layout. The core idea with other "grid" frameworks is that as devices and dimensions change, the layout can seamlessly adjust for the best viewing experience — using JavaScript or @media queries to detect changes to the viewport dimensions.

Core Concepts

Blueprint differs from these frameworks, in that it:

  1. Does not use any JavaScript
  2. Does not use a column-based system
  3. Only has 2 base files for all dimensions, rather than multiple CSS files for various dimensions, and...
  4. ...you don't need to worry about xs, s, m, lg, or different dimensional breakpoints - it handles it for you.
