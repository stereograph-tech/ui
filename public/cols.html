Introduction

Meet the .col class. This class converts the element into a column. Technically, we could have created a ".column" class...but col is shorter. In a column, children are laid out vertically, as in the example on the right.

How to use:

In our Framework, this is really easy to implement: 


    <div class='col'>

        <!-- -->
        <div class='row'>Row 1</div>
		
        ... more rows or other columns ...
        
        <!--  -->
        <div class='row'>Row N</div>
		
    </div>

How it works

Parent elements with .row applied will stretch horizontally, and each sibling row will stack vertically. Their vertical height flexes to take up the available vertical space.

Elements inside of a row without any extra styling is pretty boring, as you can see by the contents of the first row. Each example to the right will get progressively more complex.

Context

It can be tricky to work with rows on their own at the layout level if you are creating rows in the wrong context. Rows want to line things up horizontally - including themselves. To get rows to stack vertically, you need to place them inside of a column. Up next:
