List + Numpad

Introduction

The listivew + numpad view is a very basic view consisting of 3 parts:

  1. The flexible column
  2. The gutter
  3. The srink column

Example



    <section role="main" class="row">
    
        <!-- flexible column -->
        <div class="col">
        
            <!-- dynamically insert listview here -->
        
        </div>
        
        <!-- invisible column spacer (gutter) -->
        <div class="width4"></div>
        
        <!-- "shrink-wrapped" column -->
        <div class="shrink">
        
            <!-- dynamically insert numpad here -->
        
        </div>
    
    </section>

Core Concepts

.cols flex by default.

The .width4 div acts as a "gutter" between columns. This will soon be replaced by an explicit .gutter class.

this .shrink class acts as a column, but instead of flexing to take up the available space, it instead "shrinks", or "shrink-wraps" its contents. This is particularly useful when you have a fixed- width control (such as a numpad). If the width of the control changes, this alleviates the need for manually updating the column width everywhere there is a numpad; the column instead fits to the contents dynamically.

Note: The numpad's base font-size changes below 1024px and above 1600px. This is only done for illustration.
