Introduction

A panoramic view is one that is unbroken and extensive, as of a landscape. In the newest apps, this is a common layout, which involves positioning page elements in multiple columns, with horizontal scrolling to handle overflow.

How to use:

In our Framework, this is really easy to implement: 


    <section role='main' class='panoramic'>

        <!-- 40GU wide -->
        <div class='col width40'>Column 1</div>
		
        ... more columns ...
        
        <!-- 40GU wide -->
        <div class='col width40'>Column N</div>
		
    </section>

Some Considerations

Scroll to the last column in this panoramic. You might notice that it has a 6GU (grid unit) margin right. It has the marginRight6 class applied to it. However, there is a special trick used to get that margin to appear properly. This is an issue I'm currently trying to resolve. As a work around, you can click on the view source (eyeball) icon above, where you will notice find a div commented as a "Ghost Node" at the bottom.
